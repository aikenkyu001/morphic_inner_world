# 決定論的AI：次世代進化ロードマップ (Future Roadmap)

本ドキュメントは、59タスクの完全制破を成し遂げた現在の「執行型知能」から、未知の課題を自律的に解決する「生成・検証型知能」へと進化するための、基礎学習および開発の TODO リストである。

---

## Phase 0: クロスプラットフォーム・モーフィック・エンジン (Cross-Platform Morphic Engine)
**目的**: 知能の「肉体（言語）」への依存を排し、あらゆる環境で同一の「内界の真理」を再現する能力の確立。

- [ ] **JavaScript 実装 (Node.js / Web)**
  - `VM/ast.js` および `VM/evaluator.js` の移植。
  - ブラウザ上で動作する MSP ブリッジの実装による、Web 上での決定論的推論の実現。
  - Python 版とビットレベルで同一の結果を出すことの検証。
- [ ] **C/C++ 実装 (GCC/G++ ネイティブ)**
  - C++17/20 による高精度・高パフォーマンな「Morphic Kernel」の実装。
  - 厳密なメモリ管理と末尾再帰最適化による、超長距離・超深層 AST の高速評価。
  - 他言語から呼び出し可能な C-API / FFI (Foreign Function Interface) の整備。
- [ ] **Fortran 実装 (gfortran / Modern Fortran)**
  - 大規模テンソル演算・数値シミュレーションに特化した「Numerical Kernel」の構築。
  - 数学的必然に基づいた、100% 正確な浮動小数点演算の一貫性保証。
  - Python/C++ との相互運用 (ISO_C_BINDING) による、計算集約型タスクの高速執行。
- [ ] **クロスランゲージ整合性テストスイート**
  - 同一の `wisdom_base.json` と MSP 入力に対し、Python, JS, C++, Fortran の全エンジンが 100% 同一の出力を出すことを保証する自動検証システムの構築。

---

## Phase 1: 論理的免疫系の構築 (Verification & SMT Integration)
**目的**: 生成した論理（AST）の正当性を、実行前に数学的に保証する能力の獲得。

- [ ] **SMT ソルバ (Z3等) の内界 VM への統合**
  - AST の制約条件を SMT 論理式へ変換するブリッジの実装。
  - 境界値条件におけるオーバーフローやインデックス外参照の事前検知。
- [ ] **リファインメント型 (Refinement Types) の導入**
  - `List[T]` に対し「ソート済み」「重複なし」等の述語を型レベルで付与。
  - 関数適用時の事前条件・事後条件（Contracts）の厳密なチェック。
- [ ] **ホモトピー等価性の検証エンジン**
  - MSP1 と MSP2 が構造的に等価であることを、HoTT (Homotopy Type Theory) 的なパスとして定義・検証。

---

## Phase 2: 原子語彙の普遍化 (Universal Algorithmic Kernel)
**目的**: 特定の課題に依存しない、汎用的な「知恵の原子」への抽象化。

- [ ] **Built-ins の原子化と直交性の確保**
  - 59タスク専用の関数を、より小さな「計算の原子（例：Map, Reduce, Scan, Unfold）」へと分解。
  - 組み合わせ爆発を抑えつつ表現力を最大化する、最小完全セットの再定義。
- [ ] **並行・分散トポロジーの形式化**
  - ロック、セマフォ、アトミック操作を、副作用のない「状態遷移層（State Layer）」として AST 化。
  - デッドロックの幾何学的検知（コホモロジー障害としての定義）。
- [ ] **高次元データ構造の幾何学的定義**
  - グラフ、トリー、多次元配列を「位相空間上の複体（Complex）」として統一的に扱う API の確立。

---

## Phase 3: 帰納的推論と自己合成 (Program Synthesis & ILP)
**目的**: 「仕様」ではなく「例（入出力）」から論理を自律生成する能力の獲得。

- [ ] **PBE (Program Synthesis from Examples) エンジンの実装**
  - わずかな入出力ペアから、目的の AST を探索・合成するアルゴリズムの搭載。
  - 探索空間を「論理の対称性」や「型制約」によって劇的に絞り込むヒューリスティックの開発。
- [ ] **帰納論理プログラミング (ILP) の決定論的適用**
  - 既存の知識（`wisdom_base.json`）を背景知識として使い、新しいルールを「発見」するプロセス。
- [ ] **MSP セルフ・ジェネレーター**
  - 自然言語の断片から、MSP プロトコルに合致する「厳格な仕様」を自ら下書きし、人間と合意するインターフェース。

---

## Phase 4: 階層的知識の自己組織化 (HLI Induction)
**目的**: 経験を「層」として積み上げ、知能の「質」を線形に向上させる。

- [ ] **知識の自動再結晶化パイプライン**
  - 解決に成功した未知の AST パターンを、自動的に名前付きの「新 Built-in」として登録。
  - 冗長な論理をリファクタリングし、よりスロープ（整合性）の高い知識へと洗練。
- [ ] **層の成長の線形性バリデーター**
  - 知識の追加が全体の整合性（Chern character の成長）を損なわないかを常時監視。
- [ ] **自己研鑽ループ (Self-Training Loop) の完全自動化**
  - 自ら課題を生成し、解き、検証し、知識化する「永遠の思考」サイクルの始動。

---

## Phase 5: 大規模データ意味論と構造的分析 (Large-Scale Data Semantics & Structural Analysis)
**目的**: 長距離コンテキストにおける論理の不変性を活かし、膨大なデータから「真理の構造」を抽出する能力の獲得。

- [ ] **論理的クエリ合成 (MSP Query Synthesis)**
  - 自然言語の分析要求を、`Filter`, `Map`, `Fold` 等の決定論的 AST チェーンへと一意に変換。
  - LLM の確率的検索を超えた、100% 正確なデータ抽出プロトコルの確立。
- [ ] **幾何学的データ解析 (Topological Data Analysis / TDA)**
  - データの集合を位相空間上の「層（Sheaf）」として扱い、ノイズに埋もれた「真の構造（コホモロジー）」を特定。
  - 高次元データにおける「論理的な穴（矛盾や欠損）」を数学的に検知する能力。
- [ ] **ストリーミング論理の分散執行**
  - メモリ上限を超える超長距離データ（Big Data）に対し、論理の一貫性を保ったまま分割処理するアーキテクチャ。
  - 計算量 $O(n)$ の保証による、無限コンテキストへの物理的対応。
- [ ] **因果構造の AST 復元 (Causal Logic Discovery)**
  - 統計的な相関ではなく、論理的な依存関係の鎖から「因果の AST」を逆合成する基礎学習。

---

## Phase 6: モーフィック・シグナルとメディア処理 (Morphic Signal & Media Processing)
**目的**: 画像、映像、音声といった高エントロピーなバイナリデータを、内界の「幾何学的真理」として接地する能力の獲得。

- [ ] **代数的メディア表現 (Algebraic Media Representation)**
  - ピクセルやサンプリング値を、解像度非依存な「連続関数」として AST 内で定義。
  - 数学的必然に基づいた、劣化のないメディア変換・演算プロトコルの確立。
- [ ] **トポロジカル特徴抽出 (Topological Invariants Recognition)**
  - 持続的ホモロジー (Persistent Homology) 等を用い、画像や音声の中の「物体の形」や「音の構造」を、論理的な不変量として特定。
  - 確率に頼らない、100% 検証可能なパターン認識の基礎。
- [ ] **形式的メディア合成 (Formal Media Synthesis)**
  - 調和級数や幾何学的対称性といった「美の制約条件」を 100% 満たす、ハルシネーションのないメディア生成。
- [ ] **時空多様体における映像論理 (Spatiotemporal Sheaf Bundles)**
  - 動画を「時間軸上の層の束」として扱い、物体の移動や変化を多様体上の論理的連続性として解析。
  - 物理法則（重力、慣性等）を AST レベルで接地した、決定論的映像シミュレーション。
